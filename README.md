# Saudi Skills

AI skills for Saudi businesses. Integrations for local payment gateways, services, and platforms.

## Usage in Claude Code

Add this marketplace to Claude Code:

```
/plugin marketplace add khaled-harthi/saudi-skills
```

Then install individual skills:

```
/plugin install moyasar
/plugin install authentica-otp
```

## Available Skills

| Skill | Description |
|-------|-------------|
| **moyasar** | Moyasar payment gateway — cards, Mada, Apple Pay, STC Pay across web, iOS, Android, Flutter, React Native |
| **authentica-otp** | Authentica OTP verification via SMS — phone verification, 2FA, OTP-based login |
